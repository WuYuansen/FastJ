margin样式说明：（margin指的是外边距）

 .mt(上边距)   .mr(右边距)   .mb(下边距)  .ml(左边距)  .m(全边距)
 margin具体边距有（0,2,5,8,10,15,20,25,30）,例如：.mt5 .mt8 .mr5 .mr8 ......

padding样式说明：（padding指的是内边距）

 .pt(上边距)   .pr(右边距)   .pb(下边距)  .pl(左边距)  .p(全边距)
 padding具体边距有（0,2,5,8,10,15,20,25,30）,例如：.pt5 .pt8 .pr5 .pr8 ......
