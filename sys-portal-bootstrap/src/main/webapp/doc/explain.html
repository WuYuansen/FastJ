项目说明

工具和框架 (Tools and Frameworks)

Java: JDK1.7
IDE: Eclipse Indigo
Version: Spring3.2.2 / MyBatis 3.1.1 / mybatisspring 1.1.1
DataBase: MySql/oracle
Web Server: Tomcat,Jetty
Build Tool: Maven
Other: Druid 1.0.9(database connection pool) Log4j dom4j-1.6.1 FastJson1.2.6 POI3.15 ehcache-core-2.6.6 quartz-all-1.6.0等
Web Service:cfx2.7.1
		
parent_framework
---->sys_generator				系统代码生成器
---->sys_common				系统模块-公共类，Utils，邮件发送，短信模块，Excel导出，Xml解析，JSON解析，以及对MyBatis以及SpringMVC进行的封装，优化等
---->sys_dao						系统模块-DAO访问层
---->sys-tiles						系统模块-services服务层
---->sys-services					系统模块-Controller服务层以及其他
---->sys-portal-bootstrap			统一前端展示-Bootstrap、RequireJS、Angularjs版本
---->sys-portal-Extjs				统一前端展示-Extjs版本
---->其他扩展项目包				系统其他功能模块包
       	
  本系统后端采用模块化模式进行开发，每一个模块可单独作为一个服务打包加入到总服务中即可运行。如需移除某一块功能则可以直接移除对应Jar包即可完成。

  本系统除数据库连接等基础配置外无其他任何配置项。

  用户权限功能做到细粒度控制，页面，按钮，后台方法。

  系统数据缓存采用Redis作为缓存介质。

  未完待续...

框架前端(Front Frameworks)

  本系统基于Bootstrap v3.3.7、layui、AngularJS v1.5.9、requirejs进行二次开发并封装。系统展示页面开发文件存储位置必须放在pages目录下，按照模块进行存储，一个页面必须存在与之对应的一个javascript脚本文件，具体文件格式以及内容可参考系统中页面以及脚本文件。
  现在系统中封装的有：表格控件、下拉选择控件、树型控件、按钮、提示框、表单、视图、选项卡、布局控件等
